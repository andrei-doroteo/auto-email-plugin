# Auto Email WordPress Plugin

By: DoroteoDigital

Directory Structure

```
src
|
|--- Admin
|     - creates admin menu for plugin settings
|
|--- API
|     - handles REST API endpoints
|
|--- parser
|     - parses templates to populate placeholder variables
|
|--- sender
|     - sends final email as HTML
|
|--- templates
|     - email templates in HTML
```
